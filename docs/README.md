# Tomcat Docs

Tomcat 학습 문서는 주제별로 나눠 관리합니다.

| 폴더 | 내용 |
|------|------|
| `install/` | 설치와 초기 실행 |
| `architecture/` | 아키텍처, 디렉터리 구조, 클래스로더 |
| `config/` | server.xml, web.xml, context.xml, virtual host |
| `runtime/` | connector, session, JNDI, clustering |
| `integrations/` | Nginx/Apache 연동 |
| `security/` | TLS, 보안 설정 |
| `performance/` | 성능 튜닝 |
| `operations/` | 로깅, 모니터링, 실전 예제, 트러블슈팅, 운영 전략 |

처음 읽을 문서는 `install/01_installation_AL2023.md`입니다.
